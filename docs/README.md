# Commit Conventions

  + feat : 새로운 기능
  + fix : 버그 수정
  + build : 빌드 관련 파일 수정
  + chore : 그 외 자잘한 수정
  + ci : CI 관련 설정 수정
  + docs : 문서 수정
  + style : 코드 스타일 혹은 포맷 등
  + refactor : 코드 리팩토링
  + test : 테스트 코드 수정
